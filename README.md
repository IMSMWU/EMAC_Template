# EMAC_Template
A Basic EMAC LaTeX Template

## Sample

```latex
\documentclass{emac}

\begin{document}

\title{{Some fancy title}}

\ABSTRACT{%
This abstract is HUGE! (max. 1000 characters!)
}

\KEYWORDS{Keyword 1, Keyword 2, Keyword 3}
\TRACK{Your Track}

\maketitle

\section{Introduction and Research Aim}
...

\printbibliography

\end{document}
```
